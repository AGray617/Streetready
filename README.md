# StreetReady

Faith-based apologetics training platform for everyday believers in urban communities.

## Live URL
https://getstreetreadyapp.com

## Stack
- React (Create React App)
- Vercel (deployment)
- Supabase (database + auth — in progress)
- Stripe (payments — in progress)
- Anthropic API (AI Coach — proxied via Vercel serverless)
